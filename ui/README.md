## House maker minecraft

H O L A

Create houses blueprints taken from https://www.grabcraft.com/minecraft/ 

### Table of contents

* [Install](#install)
* [Usage](#usage)

### Install

#### Install dependencies

```
npm install
```
or
```
yarn
```

### Usage

#### Run the app

```
npm run start
```
or
```
yarn start
```

#### Build the app (automatic)

```
npm run package
```
or
```
yarn package
```

#### Build the app (manual)

```
npm run build
```
or
```
yarn build
```

#### Test the app (after `npm run build` || `yarn run build`)
```
npm run prod
```
```
yarn prod
```
